# 💰 Premium Expense & Income Tracker

A sleek, modern, and feature-rich Desktop Personal Finance Management Application built using **Python** and **CustomTkinter**. This app helps users take control of their financial life with real-time tracking, advanced budgeting analytics, and automated reminders.

---

## ✨ Key Features

*   **🎨 Premium Modern UI:** Built with CustomTkinter supporting dynamic Dark/Light mode switching. Includes a cinematic video background for user onboarding.
*   **💳 Multi-Account Balancing:** Live tracking for multiple accounts including Cash Wallet, Bank Account, and Credit Card.
*   **📈 Smart Financial Insights:** An intelligent rule-based engine that compares monthly spending habits and generates personalized financial advice (with localized support).
*   **📊 Budgeting Analytics (50/30/20 Rule):** Built-in dynamic data visualization using Matplotlib to track spending against the popular 50/30/20 budgeting rule.
*   **📋 Automated Recurring Bills:** A dedicated subscription manager that alerts users of upcoming bills (e.g., Netflix, Wifi) and logs approved transactions automatically.
*   **💱 Live Currency Conversion:** Integrates with exchange rate APIs to dynamically fetch and display balances in multiple global currencies ($ , €, £, ₹, ¥, Rs.).
*   **📥 Data Portability:** Supports exporting transaction history to Excel/CSV and importing raw bank statements.
*   **🔒 Secure User Auth:** Complete User Management system including secure login/registration, password reset, and a "Remember Me" local session feature.

---

## 🛠️ Tech Stack

*   **GUI Framework:** CustomTkinter & Tkinter
*   **Database:** SQLite3
*   **Data Visualization:** Matplotlib
*   **Multimedia Processing:** OpenCV (cv2) & Pillow (PIL)
*   **API Integration:** Live Exchange Rate API (JSON fetching)
